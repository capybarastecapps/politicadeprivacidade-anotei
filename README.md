Política de Privacidade - Anotei

Última atualização: 12/05/2026

O aplicativo Anotei foi desenvolvido para ser uma ferramenta de produtividade pessoal. Esta página informa sobre nossas políticas relativas à coleta, uso e divulgação de informações pessoais quando você usa nosso serviço.

1. Coleta e Uso de Informações
O Anotei foi projetado com a privacidade em mente.
•
Dados Locais: Todas as suas notas, tarefas e configurações são armazenadas exclusivamente no seu dispositivo através do armazenamento local (SharedPreferences). Nós não temos acesso, não coletamos e não armazenamos seus dados em servidores externos.
•
Biometria: Se você optar por ativar a proteção por biometria, o aplicativo utiliza as APIs oficiais do sistema operacional (Android/iOS). O Anotei não tem acesso aos seus dados biométricos (impressão digital ou rosto); o sistema apenas nos informa se a autenticação foi bem-sucedida ou não.

2. Permissões do Dispositivo
Para funcionar corretamente, o app pode solicitar:
•
Biometria/Impressão Digital: Para garantir que apenas você acesse suas notas.
•
Notificações: Para enviar lembretes de tarefas que você configurar.

3. Compartilhamento de Dados
Como seus dados são locais, o Anotei não compartilha suas informações com terceiros. O único compartilhamento ocorre quando você, por ação própria, decide compartilhar uma nota através do botão "Compartilhar" para outros aplicativos (WhatsApp, E-mail, etc.).

4. Segurança
A segurança dos seus dados é importante para nós. Como os dados estão no seu celular, a segurança deles depende também da segurança do seu dispositivo. Recomendamos o uso do recurso de bloqueio por biometria disponível no app.

5. Links para Outros Sites
Nosso serviço não contém links para outros sites, exceto se você mesmo adicionar links em suas notas.

6. Alterações nesta Política de Privacidade
Podemos atualizar nossa Política de Privacidade de tempos em tempos. Recomendamos que você revise esta página periodicamente.
