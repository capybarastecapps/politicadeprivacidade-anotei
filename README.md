# Política de Privacidade - Anotei

**Última atualização:** 03/04/2026

Esta Política de Privacidade descreve como o aplicativo **Anotei** lida com as suas informações.

## 1. Coleta de Informações
O aplicativo **Anotei** foi projetado para respeitar a sua privacidade. 
*   **Dados Pessoais:** Nós não coletamos, armazenamos ou transmitimos nenhum dado pessoal identificável (como nome, e-mail, telefone ou localização).
*   **Conteúdo do Usuário:** Todas as notas, tarefas e pastas criadas por você são armazenadas **exclusivamente no seu dispositivo local**. Nós não temos acesso ao que você escreve.

## 2. Armazenamento de Dados
O aplicativo utiliza o armazenamento local do seu dispositivo (SharedPreferences) para salvar suas anotações. Esses dados permanecem no seu telefone e não são enviados para nenhum servidor externo ou nuvem mantida por nós.

## 3. Serviços de Terceiros
O aplicativo utiliza bibliotecas padrão do Flutter para funcionalidades básicas:
*   **Notificações Locais:** Usadas apenas para lembretes configurados por você, processados inteiramente no dispositivo.
*   **Compartilhamento:** Quando você usa a função "Compartilhar", o aplicativo envia o texto da nota para o sistema operacional do seu celular para que você escolha o destino.

## 4. Permissões
O aplicativo pode solicitar permissões básicas para o funcionamento correto:
*   **Notificações:** Para exibir lembretes de tarefas.

## 5. Exclusão de Dados
Você tem total controle sobre seus dados. 
*   Você pode apagar notas individualmente.
*   Você pode usar a função "Limpar Tudo" nas configurações do app para remover todos os dados locais.
*   Ao desinstalar o aplicativo, o sistema operacional Android geralmente remove todos os dados associados a ele.

## 6. Alterações nesta Política
Podemos atualizar nossa Política de Privacidade de tempos em tempos. Recomendamos que você revise esta página periodicamente para quaisquer alterações.
